Bonjour,

Je m'appelle Antoine Grévin, je suis étudiant de première année en réseaux et télécommunications à l’Institut Universitaire de Technologie de Béthune compris dans l’Université d’Artois.

Semestre 1

SAE1.01 - Se sensibiliser à l'hygiène informatique et à la cybersécurité

J’ai suivi le MOOC proposé par l’ANSSI et le cours "Introduction à la cybersécurité" sur Netacad. Ces cours m’ont permis d’acquérir des notions sur les risques numériques, les bonnes pratiques de sécurité et les contre-mesures adaptées. J’ai obtenu les attestations de réussite pour ces deux formations.

Compétences acquises : bonnes pratiques, risques numériques, mesures de protection, communication adaptée et utilisation des outils numériques.

Objectif : se mettre dans la peau d’un Responsable de la Sécurité des Systèmes d’Information (RSSI) et sensibiliser le personnel aux risques numériques.

Difficultés : aucune particulière.

Auto-évaluation : A B C D

SAE1.02 - S'initier aux réseaux informatiques

L’objectif est de mettre en œuvre un réseau pour comprendre ses interactions. Nous avons conçu, configuré et testé l’infrastructure réseau, avec VLAN, services réseau (DNS, DHCP, serveur web) et plan d’adressage IPv4.

Difficultés rencontrées : certains VLAN ne communiquaient pas correctement au début, nécessitant une révision des configurations sur les Switch.

Compétences acquises : conception et configuration de VLAN, plan d’adressage IPv4, installation de services réseau, utilisation de Packet Tracer, draw.io, Wireshark, et systèmes Linux.

Durée : 1 mois, 2 semaines et 1 jour.

Auto-évaluation : A B C D

SAE1.03 - Découvrir un dispositif de transmission

Découverte et utilisation de Matlab Simulink pour analyser et concevoir des systèmes de transmission.

Difficultés : erreurs initiales dans Matlab entraînant des signaux incorrects.

Compétences acquises : utilisation de Matlab/Simulink, analyse et conception de systèmes, maîtrise de plusieurs types de signaux.

Durée : environ 1 mois.

Auto-évaluation : A B C D

SAE1.04 - Se présenter sur Internet

Réalisation d’un site web responsive respectant les standards W3C.

Compétences acquises : maîtrise de HTML/CSS, utilisation d’un framework CSS, création et gestion de contenu dynamique.

Difficultés : aucune particulière.

Durée : 4 séances de 4 heures + présentation finale.

Auto-évaluation : A B C D

SAE1.05 - Traiter des données

Récupération, transformation et transmission de données sur un serveur web de façon automatisée.

Difficultés : données inexploitables ou incorrectes, problèmes lors des transformations, gestion du temps.

Compétences acquises : gestion et traitement de données, codage, collaboration et documentation.

Durée : environ 3 semaines.

Auto-évaluation : A B C D

Semestre 2

SAE2.01 - Construire un réseau informatique pour une petite structure

Difficultés : configuration des équipements réseau et services comme les partages entre OS différents.

Compétences acquises : conception, déploiement et sécurisation d’infrastructure réseau complète, configuration système, administration réseau, Docker, sécurité, travail collaboratif.

Durée : 1 semaine.

SAE2.02 - Mesurer et caractériser un signal ou un système

Analyse et caractérisation de signaux avec outils matériels et logiciels.

Difficultés : conception de filtres numériques et interprétation des réponses fréquentielles, utilisation des outils logiciels.

Compétences acquises : analyse spectrale, utilisation d’analyseurs de spectre, conception de filtres numériques, traitement et interprétation des signaux.

Durée : plusieurs semaines.

Auto-évaluation : A B C D

SAE2.03 - Mettre en place une solution informatique pour l'entreprise

Création d’un site portfolio dynamique avec Flask, base de données et déploiement via Docker.

Difficultés : configuration de Docker pour communication entre conteneurs, adaptation du docker-compose.yml pour stabilité.

Compétences acquises : conception d’applications web dynamiques, modélisation BDD, interactions app-base, déploiement Docker.

Durée : plusieurs semaines.

Auto-évaluation : A B C D

SAE2.04 - Projet intégratif

Conception et mise en œuvre d’une infrastructure réseau complète avec planification, configuration routeurs/switches, services web, bases de données, téléphonie IP et virtualisation via Docker.

Compétences acquises : administration réseau, configuration équipements, virtualisation Docker, sécurisation, gestion services web et téléphonie IP, IoT et vidéosurveillance, planification et travail collaboratif.
